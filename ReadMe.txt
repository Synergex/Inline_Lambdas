Description:
	This program is a simple example of the syntax and
	 usefulness of using Inline Lambdas
		

To build and run the program:
	Lambdas are a .NET only feature.
	The code could either be copied into a Synergy .NET console 
	project in Visual Studio or run at the command line with the below - 

	
	path=%path%;C:\Program Files (x86)\MSBuild\Synergex\dbl
	dblnet inlinelambda
